Tools for downloading sports statistics.


.. highlight:: shell

Installation
------------

1. Install pip::
   
   $ sudo easy_install pip
   
or follow PyPI's installation_ guide

2. Install `sportsstats`:: 
   
   $ pip install sportsstats

Usage
-----

Run: ``stats -h`` for usage info and help.
