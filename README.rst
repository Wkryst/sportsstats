Tools for downloading sports statistics.

* Free software: ISC license
* Documentation: https://sportsstats.readthedocs.org.

Installation
------------


1. ``sudo easy_install pip``

2. ``pip install sportsstats``

Usage
-----

To see usage info, run: ``stats -h``
